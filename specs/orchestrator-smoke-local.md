# Local Orchestrator Smoke Test

This is a tiny live test for the Pi orchestrator extension in this repo.

Do not modify extension source code.
Do not merge the integration branch back to main.

## Worker 1: hello-file

Create `tmp-orchestrator-smoke/hello.txt` with exactly this content:

```text
hello from orchestrator
```

Read the file back and mention the result in your worker summary.

## Worker 2: package-summary

Read the root `package.json`.

Create `tmp-orchestrator-smoke/package-summary.md` containing:

- the package name
- the package version
- a sentence saying it was generated by the package-summary worker

Read the file back and mention the result in your worker summary.

## Integration

After worker branches are merged into the integration worktree, verify:

- `tmp-orchestrator-smoke/hello.txt` exists
- `tmp-orchestrator-smoke/package-summary.md` exists
- `tmp-orchestrator-smoke/hello.txt` contains `hello from orchestrator`

Finally, clean up orchestrator worktrees. Do not merge integration back to main.
